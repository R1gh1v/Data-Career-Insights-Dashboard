# Data Professional Survey Analysis

## Project Overview
This project provides a comprehensive look into the data science and analytics profession based on global survey data. The goal was to clean raw survey responses and create an interactive dashboard that helps aspiring and current data professionals understand industry trends regarding compensation, work-life balance, and technical requirements.

## Key Insights Explored
* **Salary Distribution**: Average yearly salary broken down by job title.
* **Top Programming Languages**: Identifying the most popular languages among professionals (Python, R, SQL, etc.).
* **Job Satisfaction**: Metrics on how happy professionals are with their salary, management, and work-life balance.
* **Demographics**: Breakdown of professionals by country and education level.
* **Market Entry**: Analysis of how difficult it is to break into the data field.

## Tools Used
* **Power BI**: For data modeling, DAX calculations, and visualization.
* **Excel / Power Query**: For initial data cleaning, handling nulls, and standardizing country names and job titles.

## How to View the Project
1. Download the `Final Project.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. If the data source path needs updating, simply point it to the included `Power BI - Final Project.xlsx - Data Professional Survey.csv` file.
